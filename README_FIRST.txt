Legacy Core / Legacy Wallet - Windows Release

1. Extract the ZIP to any folder.
2. Double-click START_HERE.bat to launch Legacy Wallet.
3. Keep all files together — the wallet needs the bundled DLLs.

First launch:
- Windows may ask for firewall permission for P2P port 19555 — allow it
- Wallet data: %APPDATA%\LegacyCoin
- RPC port 19556 should stay private

Security:
- Back up your wallet before receiving or mining funds
- Never share backups, keys, seed phrases, or RPC credentials
- Only download releases from https://github.com/legacybtc/LegacyCore
- Verify checksums against SHA256SUMS.txt before running
